方法一：

解压压缩包，将解压后文件夹移动到你放拓展的文件夹，然后浏览器选择【加载解压缩的拓展】

注意：
由于这个没有封装成chrome拓展，所以可能会出现过段时间点击拓展提示找不到文件，这个时候重新【加载解压缩的拓展】即可。解决办法只有等原作者更新代码打包发布

方法二：

使用打包好的crx拓展，浏览器拓展管理页面，点击打包拓展，扩展根目录选择压缩包解压后的文件夹。然后拖动打包好的crx文件到浏览器。
因为是本地安装，edge会提示不能使用，教程见https://www.bilibili.com/read/cv10090713/
若后续需要更新本地安装的拓展，需要用到pem后缀的密钥文件
